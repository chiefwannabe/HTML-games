## ✨ Features

- 🎯 **Lightweight & Fast** — No external dependencies, minimal load times
- 📱 **Responsive Design** — Play on desktop, tablet, or mobile
- 🎨 **Clean UI** — Intuitive interfaces with visual feedback
- ⌨️ **Keyboard & Touch Support** — Works with keyboard and touch events
- 🔄 **State Management** — Score tracking, game resets, and persistent state
- 🎵 **Sound Effects** (optional) — Simple Web Audio API sounds
- 🌐 **Offline Play** — No internet connection required after first load
